# Credits and Third-Party Licenses

This project incorporates code and assets from various open-source projects. We are grateful to the following creators and communities for their contributions:

### External Projects and Assets

*   **Godot Engine**: The game engine powering this project (MIT License).
*   **Decentraland**: LiveKit Rust extension architecture inspired by Decentraland's open-source implementations.
*   **TLabWebViewVR**: Browser extension functionality adapted from TLabWebViewVR and Unity XR Interaction Toolkit to supplement Godot's WebView capabilities.
*   **Justin Barnett**: Physics-based hand interaction and PID controller architecture based on Justin Barnett's VR research and implementations.
*   **Norodix**: Copyright (c) 2022 Norodix for the mirror implementation.
*   **World Grab Demo**: The "Simple World Grab" implementation is based on the World Grab demo projects, providing a robust foundation for VR world manipulation.
*   **Godot XR Tools**: Various XR interaction patterns, pointer rigs, and scripts are derived from the Godot XR Tools library.
*   **Godot Jolt Physics**: High-performance physics engine for Godot via the Godot Jolt extension.
*   **LiveKit**: Networking infrastructure for real-time video, audio, and data.
*   **Nakama**: Distributed server for social and real-time games and apps.
*   **Procedural Materials & Shaders**: Materials used in the tools demo, including grass, sand, marble, ice, and lava effects, are created through procedural generation and custom shaders.

### Services and Infrastructure

*   **Oracle Cloud**: Cloud hosting and server infrastructure.
*   **Sentry**: Error tracking and performance monitoring.
